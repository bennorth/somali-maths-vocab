# Somali / English maths vocabulary

Data extracted and refined from:

- Study Companion, Word List / Eraybixinta, Mathematics / Xisaab, English–Somali / Soomaali–Ingiriis, HAAN Associates 1992.

## Build instructions

For deployment to GitHub pages:

Create a git worktree with working directory `dist` checked out to
branch `dist`.

Within project root, build with:

``` shell
npm run build -- --base=/somali-maths-vocab/
```

which updates files in `dist/`.  Push the `dist` branch to GitHub.
